Analysis of Algorithms

Overview

  1. An algorithm is a set of instructions to perform a task or to solve a given problem.
  2. There are several different algorithms to solve a given problem.
  3. The analysis of algorithms involves finding the best algorithm that runs fast and uses less memory.

For Example:

Question: Find the sum of the first n natural numbers.

a) Input: n = 4

Output: 10 (i.e., 1 + 2 + 3 + 4)

b) Input: n = 5

Output: 15 (i.e., 1 + 2 + 3 + 4 + 5)
