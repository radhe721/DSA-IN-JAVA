First Algorithm Explanation

The first method, findSum(int n), uses the mathematical formula to calculate the sum of the first n natural numbers:

n * (n + 1) / 2

For example, if n is 5:

  • Compute n * (n + 1): 5 * 6 = 30
  • Divide by 2: 30 / 2 = 15

Thus, the sum of the first 5 natural numbers is 15.

Second Algorithm Explanation

The second method, FindSum(int n), calculates the sum using a loop:

  • Initialize sum to 0.
  • Iterate from 1 to n, adding the current number to sum.
  • Return sum.

For example, if n is 5:

  • Sum starts at 0.
  • Add 1: 0 + 1 = 1
  • Add 2: 1 + 2 = 3
  • Add 3: 3 + 3 = 6
  • Add 4: 6 + 4 = 10
  • Add 5: 10 + 5 = 15

Thus, the sum of the first 5 natural numbers is 15.
